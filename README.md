\# HealthWell - Student Wellness Platform



\## Project Structure

\- `/frontend` - React + Vite application

\- `/backend` - Spring Boot REST API



\## Tech Stack

\### Frontend

\- React 19, Vite, Tailwind CSS, React Router, Axios



\### Backend

\- Spring Boot, REST APIs, MySQL



\## Running Locally



\### Frontend

```bash

cd frontend

npm install

npm run dev

```



\### Backend

\- Open `backend` folder in Eclipse

\- Run as Spring Boot App

\- Runs on `http://localhost:8080`
